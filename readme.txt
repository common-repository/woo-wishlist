WooCommerce Wishlist is a plugin to add a customizable wishlist to your site.

== Description ==

Looking for a cool design to create a wishlist and add products to it?

Here is the perfect solution. With WooCommerce Wishlist, you can save as many products as you like to your wishlist with just one click and view each product added instantly on a fixed wishlist box.

You can save your list for future purchases. Each user can have their own wishlist in which they can store their favorite products and access them any time they like. Of course, you will need to log in before you can access your wishlist.

**WooCommerce Wishlist Features:**

* Options to let your customers create their own wishlist.
* Add a shortcode on any page to display the wishlist.
* Change the wishlist icon shape and color as per your requirement.
* Options to position the add to wishlist icon on each product.
* Options to change the wishlist icon to text.
* Options to edit the title of your wishlist.

Additionally, we have a feature that displays a fixed box with the list of all added products. You can position the box in 6 different locations on your site.

== Installation ==

1. Upload "WooCommerce WishList" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Installation complete.
